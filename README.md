<h1 align=center>A simple gulp assembly</h1>

This is short version based on global nodejs variables and "require-dir" pugin.

## Description

Gulp assembly work with:

- pug
- html
- js
- css
- scss
- font
- image

You can use two modes:

- production mode
- developer mode
